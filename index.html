Movie React App


The project uses React/Redux to create a movie recommendation system. Users can view movies based on genre or popularity. A user can create an account to leave movie reviews or add movies to their watchlist. There is an "onboarding" for logged-in users to explain the system. Google's Firebase is used for account storage. The movie list is limited since the project is not connected to an external movie API.
